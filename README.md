# 📊 AI Analytics Copilot

An AI-powered analytics dashboard built with **Streamlit**, **Pandas**, **Plotly**, and **Google Gemini** that helps users explore datasets, generate AI-driven insights, and interact with their data using natural language.

---

## 🚀 Features

- 📂 Upload CSV or Excel datasets
- 👀 Dataset Preview
- 📈 KPI Dashboard
  - Total Rows
  - Total Columns
  - Missing Values
  - Duplicate Rows
- 📊 Summary Statistics
- 📝 Data Types Overview
- 📉 Visualizations
- 🤖 AI-Generated Dataset Insights using Google Gemini
- 💬 Chat with Your Dataset
- 📥 Download AI Insights Report

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/AI-Analytics-Copilot.git
```

Move into the project folder:

```bash
cd AI-Analytics-Copilot
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it.

**Windows**

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```
